# BookSpread-
BookSpread es una herramienta web gratuita y profesional que permite crear archivos PDF listos para impresión en formato libro (doble página horizontal). 

No requiere instalación, servidores ni compilación. **Abre y usa.**

---

## 🧪 Casos de uso reales

| Situación | Solución con BookSpread |
|-----------|-------------------------|
| Necesito un PDF con 20 fotos para un álbum de bodas | Carga 10 hojas, coloca fotos izquierda/derecha, exporta |
| La imprenta pide márgenes de 1.5 cm interior y 1 cm exterior | Configura los márgenes exactos en la interfaz |
| Quiero marcar el doblez para que el operario lo vea | Activa la línea punteada y aparecerá en el PDF |
| Las imágenes tienen diferentes proporciones | Elige "Contener" o "Cubrir" según prefieras |

---

## ⚠️ Limitaciones conocidas

- La herramienta funciona 100% en el navegador, no guarda los proyectos (puedes guardar el HTML localmente)
- Para documentos con muchas imágenes de alta resolución, la exportación puede tomar unos segundos
- El PDF generado no es editable posteriormente (es el producto final)
